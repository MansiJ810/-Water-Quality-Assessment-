# -Water Quality Assessment-
This project evaluates water safety by leveraging machine learning algorithms to analyze critical quality indicators like pH, hardness, dissolved solids, and chemical concentrations

# Project Overview
Ensuring access to clean drinking water is vital. This project processes water testing datasets to clean missing values, handle data imbalances, and train robust classification models. The insights generated aim to improve water safety monitoring and provide actionable data for resource management.

# Tech Stack
* Language: Python
* Libraries: pandas, numpy, scikit-learn, matplotlib, seabornModels
* Evaluated: Random Forest, Logistic Regression, Decision Trees

# Model Performance
We evaluated various models based on Accuracy, Precision, and Recall to minimize false negatives (predicting unsafe water as safe). The Random Forest Classifier achieved the highest overall accuracy of 89%.
